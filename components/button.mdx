## Overview

`Button` is a flexible, accessible button component built on Base UI's `useRender` hook. It supports multiple visual variants and sizes, and can render as any element via the `render` prop.

## Installation

```bash
npx shadcn@latest add @coss/button
```

## Basic usage

```tsx
import { Button } from "@/components/ui/button"
```

```tsx
<Button>Click me</Button>
```

## Variants

The `variant` prop controls the visual style of the button.

| Variant               | Description                                            |
| --------------------- | ------------------------------------------------------ |
| `default`             | Primary action button with filled background           |
| `secondary`           | Secondary action with neutral background               |
| `outline`             | Bordered button with transparent background            |
| `ghost`               | No border or background, subtle hover state            |
| `link`                | Styled as an underlined hyperlink                      |
| `destructive`         | Filled red button for destructive actions              |
| `destructive-outline` | Bordered red button for destructive actions            |

## Sizes

The `size` prop controls the button dimensions.

| Size      | Description                              |
| --------- | ---------------------------------------- |
| `xs`      | Extra-small height (28px / 24px on sm)   |
| `sm`      | Small height (32px / 28px on sm)         |
| `default` | Default height (36px / 32px on sm)       |
| `lg`      | Large height (40px / 36px on sm)         |
| `xl`      | Extra-large height (44px / 40px on sm)   |
| `icon`    | Square icon button (36px / 32px on sm)   |
| `icon-xs` | Extra-small square icon (28px / 24px)    |
| `icon-sm` | Small square icon (32px / 28px on sm)    |
| `icon-lg` | Large square icon (40px / 36px on sm)    |
| `icon-xl` | Extra-large square icon (44px / 40px)    |

## Props

<ParamField path="variant" type='"default" | "secondary" | "outline" | "ghost" | "link" | "destructive" | "destructive-outline"'>
  The visual style variant. Defaults to `"default"`.
</ParamField>

<ParamField path="size" type='"xs" | "sm" | "default" | "lg" | "xl" | "icon" | "icon-xs" | "icon-sm" | "icon-lg" | "icon-xl"'>
  The size of the button. Defaults to `"default"`.
</ParamField>

<ParamField path="render" type="React.ReactElement">
  Render the button as a different element, such as a `Link` component. When set, the `type` attribute is omitted automatically.
</ParamField>

<ParamField path="...props" type="React.ButtonHTMLAttributes<HTMLButtonElement>">
  All standard button HTML attributes are supported.
</ParamField>

### Render as link

Use the `render` prop to render the button as a different element, such as a Next.js `Link`.

```tsx
import Link from "next/link"
import { Button } from "@/components/ui/button"

<Button render={<Link href="/login" />}>Login</Button>
```
