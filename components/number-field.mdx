## Overview

`NumberField` wraps Base UI's `NumberField` primitive family. It combines a text input with increment and decrement buttons, and optionally a `NumberFieldScrubArea` that lets users click-drag on a label to change the value.

## Installation

```bash
npx shadcn@latest add @coss/number-field
```

## Basic usage

```tsx
import {
  NumberField,
  NumberFieldDecrement,
  NumberFieldGroup,
  NumberFieldIncrement,
  NumberFieldInput,
  NumberFieldScrubArea,
} from "@/components/ui/number-field"
```

```tsx
<NumberField defaultValue={0}>
  <NumberFieldScrubArea label="Quantity" />
  <NumberFieldGroup>
    <NumberFieldDecrement />
    <NumberFieldInput />
    <NumberFieldIncrement />
  </NumberFieldGroup>
</NumberField>
```

### NumberField

<ParamField path="defaultValue" type="number | null">

</ParamField>

<ParamField path="min" type="number">

</ParamField>

<ParamField path="max" type="number">
  Maximum allowed value.
</ParamField>

<ParamField path="step" type="number">

</ParamField>

</ParamField>

  Prevents interaction.
</ParamField>

<ParamField path="...props" type="NumberFieldPrimitive.Root.Props">
  All props from Base UI's `NumberField.Root` are supported.
</ParamField>

### NumberFieldGroup

Container for `NumberFieldDecrement`, `NumberFieldInput`, and `NumberFieldIncrement`.

### NumberFieldInput

The text input that displays and accepts the numeric value.

### NumberFieldDecrement

Button that decrements the value by one step. Renders a minus icon.

### NumberFieldIncrement

Button that increments the value by one step. Renders a plus icon.

### NumberFieldScrubArea

A clickable, draggable label. Dragging left decrements and dragging right increments the value. Must be used inside a `NumberField`.
