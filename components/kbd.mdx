## Overview

The `Kbd` component renders a single keyboard key with a styled badge appearance. For multi-key combinations like `Cmd+K`, wrap multiple `Kbd` elements in `KbdGroup`. Both components are purely presentational and accept standard HTML `<kbd>` attributes.

## Installation

```bash
npx shadcn@latest add @coss/kbd
```

## Basic usage

```tsx
import { Kbd, KbdGroup } from "@/components/ui/kbd"

// Single key
<Kbd>K</Kbd>

// Key combination
<KbdGroup>
  <Kbd>⌘</Kbd>
  <Kbd>K</Kbd>
</KbdGroup>
```

## Sub-components

| Component | Description |
| --- | --- |
| `Kbd` | Renders a single keyboard key. Styled as a small muted badge. |
| `KbdGroup` | Wraps multiple `Kbd` elements for a key combination. Renders an inline `<kbd>` with a gap between keys. |

## Props

### Kbd

<ParamField path="className" type="string">
  Additional CSS classes applied to the `<kbd>` element.
</ParamField>

Accepts all standard `React.ComponentProps<"kbd">` attributes.

### KbdGroup

<ParamField path="className" type="string">
  Additional CSS classes applied to the group `<kbd>` element. Use negative margin (e.g. `-me-1`) to visually align the group inside a button.
</ParamField>

Accepts all standard `React.ComponentProps<"kbd">` attributes.

## Examples

### Inside a button

Use a negative end margin on `KbdGroup` to maintain visual balance inside a button.

```tsx
import { Button } from "@/components/ui/button"
import { Kbd, KbdGroup } from "@/components/ui/kbd"
import { PrinterIcon } from "lucide-react"

export function ButtonWithShortcut() {
  return (
    <Button variant="outline">
      <PrinterIcon aria-hidden="true" />
      Print
      <KbdGroup className="-me-1">
        <Kbd>⌘</Kbd>
        <Kbd>P</Kbd>
      </KbdGroup>
    </Button>
  )
}
```

### In a command item

```tsx
import {
  Command,
  CommandEmpty,
  CommandInput,
  CommandItem,
  CommandList,
  CommandPanel,
  CommandShortcut,
} from "@/components/ui/command"
import { Kbd, KbdGroup } from "@/components/ui/kbd"

const items = [
  { value: "new-file", label: "New File" },
  { value: "open", label: "Open" },
  { value: "save", label: "Save" },
]

export function CommandWithKbd() {
  return (
    <CommandPanel className="w-full max-w-sm">
      <Command items={items}>
        <CommandInput placeholder="Search actions..." />
        <CommandEmpty>No results.</CommandEmpty>
        <CommandList>
          {(item) => (
            <CommandItem key={item.value} value={item.value}>
              {item.label}
              <CommandShortcut>
                {item.value === "new-file" && (
                  <KbdGroup><Kbd>⌘</Kbd><Kbd>N</Kbd></KbdGroup>
                )}
                {item.value === "open" && (
                  <KbdGroup><Kbd>⌘</Kbd><Kbd>O</Kbd></KbdGroup>
                )}
                {item.value === "save" && (
                  <KbdGroup><Kbd>⌘</Kbd><Kbd>S</Kbd></KbdGroup>
                )}
              </CommandShortcut>
            </CommandItem>
          )}
        </CommandList>
      </Command>
    </CommandPanel>
  )
}
```
