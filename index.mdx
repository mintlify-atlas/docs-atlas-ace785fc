## The everything-but-AI UI stack

**coss ui** is a collection of beautifully designed, accessible, and composable components for your React apps. Built on top of [Base UI](https://base-ui.com/) and styled with [Tailwind CSS](https://tailwindcss.com/), it's designed for you to copy, paste, and own.

We're building the component library that powers [Cal.com](https://cal.com) — in the open, for everyone.

## How it works

Our approach is simple: **you should own your code.** Inspired by the copy-paste ethos of shadcn/ui, you get the source — no black-box package abstractions.

<CardGroup cols={3}>
  <Card title="Primitives" icon="layers">
    Low-level accessible building blocks powered by Base UI. Handle a11y, keyboard navigation, and focus management without dictating design.
  </Card>
  <Card title="Particles" icon="atom">
    Pre-assembled components — forms, tables, date pickers — that combine multiple primitives into ready-to-use patterns.
  </Card>
  <Card title="Atoms" icon="bolt">
    API-enhanced particles that connect UI to external services. Think scheduling, auth flows, and more — out of the box.
  </Card>
</CardGroup>

## Quick install

<CodeGroup>
```bash New project
npx shadcn@latest init @coss/style
```

```bash Add to existing project
npx shadcn@latest add @coss/ui
```

```bash Add theme tokens
npx shadcn@latest add @coss/style
```
</CodeGroup>

<Note>
  coss ui requires a React project with **Tailwind CSS v4**. See the [Get Started](/get-started) guide for full setup instructions.
</Note>
